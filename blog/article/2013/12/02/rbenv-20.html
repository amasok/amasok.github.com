対処

OpenSSLのライブラリですがせっかくなのでCenctOSとUbuntu両方乗っけときます。

CentOS

1
$ sudo yum -y install openssl-devel

Ubuntu

1
$ sudo apt-get -y install libssl-dev

これでエラー解消です。 ちょこっと焦りました。
